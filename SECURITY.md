# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.2.x   | Yes       |
| 0.1.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly by emailing **security@ux2.dev**. Do not open a public issue.

You can expect an initial response within 48 hours. We will work with you to understand the issue and coordinate a fix before any public disclosure.
